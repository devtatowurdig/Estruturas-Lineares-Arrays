# 📚 Estruturas Lineares — Exercícios com Vetores (Arrays)

Repositório contendo a resolução dos exercícios propostos na disciplina de Estruturas Lineares, com foco em manipulação de vetores (arrays) utilizando lógica manual, sem uso de métodos de alto nível.

📄 Base dos exercícios: material fornecido em aula.

---

## 🎯 Objetivo

Praticar conceitos fundamentais de estruturas lineares, especialmente:

- Manipulação de arrays com tamanho fixo  
- Deslocamento de memória  
- Busca e acesso a dados  
- Simulação de vetores dinâmicos  
- Algoritmos de otimização e merge  

> ⚠️ Restrição importante: evitar métodos prontos como `.sort()` ou operações que alterem automaticamente o tamanho do array.

---

## 🛠️ Tecnologias Utilizadas

- JavaScript  
- Node.js (para execução local)

---

## 🧠 Conceitos Trabalhados

- Complexidade de algoritmos (O(1), O(n))  
- Manipulação manual de memória  
- Estruturas de dados lineares  
- Lógica de algoritmos
